🚀 如何部署這個儀表板

1

Vercel（推薦・最簡單）

到 vercel.com 註冊，把這個 HTML 檔放進 GitHub repo，點 Import → 自動部署。完成後你會得到一個 https://你的名字.vercel.app 的免費網址。

2

Netlify Drop（拖拉即部署）

到 app.netlify.com/drop，直接把這個 HTML 檔案拖進去，10 秒即上線。

3

GitHub Pages（免費）

建一個 repo → 放入此檔並命名為 index.html → Settings → Pages → Deploy from main branch。

💡 若要定期更新數據，可搭配 GitHub Actions 定時抓取 FRED API 數據並自動更新此頁面。
